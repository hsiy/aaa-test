makeReports.views.helperFunctions.todos.section1ToDo(report)¶

Generates the ToDo list for section 1 and first page of report, includes things missing from the beginning of the report

Parameters

report (Report) – in-progress report to generate to-do for

Returns

dictionary of to-dos, QuerySet of SLOs in report

Return type

dict, QuerySet

Notes

Extra return values prevent repeatedly querying the database for the same things
